# 商城后台管理系统 java616

## 项目概述

商城后台管理系统是一个基于Vue和Node.js的前后端分离项目，旨在为用户提供一个高效、易用的管理平台。该项目包括源码和数据库，支持多种开发环境。

## 技术栈

- 前端：Vue、ElementUI
- 后端：Node.js、axios、koa
- 数据库：MySQL

## 功能模块

- **登录模块**：用户登录鉴权功能
- **用户管理**：管理系统用户
- **角色管理**：定义和分配用户角色
- **商品管理**：商品信息增删改查
- **订单管理**：处理订单相关信息

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 支持的IDE：vscode、idea、webstorm、hbuilder

## 目录结构

```
/(项目根目录)
|-- /src                # 源码目录
|   |-- /assets         # 资源文件
|   |-- /components     # Vue组件
|   |-- /router         # 路由配置
|   |-- /store          # 状态管理
|   |-- /views          # 页面视图
|-- /config             # 配置文件
|-- /database           # 数据库相关
|-- /public             # 公共文件
|-- package.json        # 项目描述文件
```
